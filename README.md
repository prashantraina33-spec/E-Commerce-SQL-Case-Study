# 🛒 E-Commerce SQL Case Study

## 📌 Overview
This project analyzes an e-commerce dataset using MySQL to extract insights on customers, products, sales, and revenue.

---

## 🎯 Objectives
- Identify top customers
- Analyze product performance
- Track sales trends
- Improve business decision-making

---

## 🗂️ Database Tables
- Customers
- Orders
- Products
- OrderDetails

---

## 🛠️ Tools Used
- MySQL
- SQL (JOINs, GROUP BY, Aggregations)

---
